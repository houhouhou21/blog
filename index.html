实验题目：利用云平台搭建个人博客
实验学时：2 实验日期：2023/3/21
实验目的：熟悉个人博客系统的搭建。
具体包括：
参考方案：注册Github账号，搭建Hexo环境并实现个人博客搭建，撰写实验报告。

硬件环境：
联网的计算机一台

软件环境：
Windows

实验步骤与内容：

  1. 注册一个Github账号，并且创建一个新的repository,用于存储hexo博客的源码和配置文件

  2. 在本地电脑上安装git和node.js环境

  3. 使用npm命令安装hexo命令行工具

  4. 使用Hexo命令行工具初始化博客源码，可以使用以下命令初始化：

hexo init blog
cd blog
npm install

  1. 使用hexo命令行工具初始化博客代码

  2. 将博客源码上传到GitHub仓库中。可以使用以下命令将代码提交到GitHub仓库：
    git add .
    git commit -m “Initial commit”
    git remote add origin
    git push -u origin master

  3. 在GitHub仓库中设置Pages服务，将博客源码转换为静态网站并发布到GitHub Pages上。可以在GitHub仓库的Settings页面中进行设置。

  4. 修改blog的配置文件，执行hexo generate命令生成静态网站，并使用hexo deploy命令将网站部署到github上。

结论分析与体会：

hexo需要使用npm命令进行安装，使用hexo的各种命令可以生成静态网站，并且将静态网站部署到github pages上。
链接的过程中容易出现无法连接的问题，要解决连接问题，可以使用ssh.
